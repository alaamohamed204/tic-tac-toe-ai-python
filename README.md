# Tic Tac Toe AI Game
A graphical Tic Tac Toe game built using Python and Tkinter, featuring an AI opponent using the Minimax algorithm with Alpha-Beta pruning.
## Features
- Player vs AI mode
- AI using Minimax algorithm
- Graphical interface using Tkinter
- Game reset functionality
- Win/Lose detection
## Technologies Used
- Python
- Tkinter
- Minimax Algorithm (AI)
